需要建立在搭建好Fabric之上的机器上,这里有讲.==>戳我

进入到caliper-benchmarks目录下,依次运行以下命令

npm i @hyperledger/caliper-cli

npx caliper --version

npm init -y

npm install --only=prod @hyperledger/caliper-cli

npx caliper bind --caliper-bind-sut fabric --caliper-bind-sdk 1.4.0(过程较长,稍等)

去net 里面的config_solo里面,运行以下generate.sh,如下命令
./generate.sh

npx caliper launch manager \ --caliper-workspace . \ --caliper-benchconfig benchmarks/scenario/simple/config.yaml \ --caliper-networkconfig networks/fabric/v1/v1.4.1/2org1peergoleveldb/fabric-go.yaml

剩下的修改可以去caliper-benchconfig benchmarks/scenario/simple/config.yaml这个目录下修改TPS即可
